# Collatz Variant Explorer

An interactive visualization tool for exploring parameterized sequence rules related to the Collatz conjecture.

## 🔗 Live Demo

**[Launch the Explorer](https://ohiomathteacher.github.io/collatz-explorer/)**

## Features

- **Interactive Graph Visualization**: Watch sequences unfold as animated node networks
- **Adjustable Parameters**: Control m, a, and d to create different sequence variants
- **Batch Testing**: Test multiple starting values at once (e.g., `1-20` or `1,5,10-20`)
- **Click-to-Inspect**: Click any node to see detailed information
- **Data Table View**: Toggle a table showing all tested sequences and outcomes
- **AI Thinking Partner**: Optional integration with AI to help analyze patterns (requires OpenRouter API key)

## The Rule

The explorer visualizes sequences generated by:

- **f(n) = n/d** when n is even
- **f(n) = mn + a** when n is odd

Where m, a, and d are adjustable parameters.

## How to Use

1. **Adjust parameters** using the sliders (when unlocked)
2. **Add starting values**:
   - Single: Type a number and click "Add"
   - Batch: Type ranges like `1-20` or lists like `1,5,10-20` and click "Batch"
3. **Interact with the graph**:
   - Watch nodes animate and connect
   - Click nodes to inspect them
   - Yellow glowing nodes are in cycles
   - Blue glowing nodes are your starting values
4. **View data**: Click the 📊 Table button to see all results organized
5. **Use AI** (optional): Get a free API key from [openrouter.ai](https://openrouter.ai) and paste it into the chat panel

## For Educators

This tool is designed for mathematical inquiry. Students can:

- Discover patterns in sequence behavior
- Form and test conjectures
- Compare different parameter sets
- Develop critical thinking about computational exploration

## Technical Details

- Built with React (via CDN)
- Pure client-side JavaScript - no server required
- AI chat uses OpenRouter API (DeepSeek model)
- Canvas-based force-directed graph rendering

## Privacy & Security

- Your API key (if entered) is stored only in your browser's localStorage
- No data is sent anywhere except to OpenRouter when you use the chat feature
- The applet runs entirely in your browser
